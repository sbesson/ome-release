  • Mac users, please note that in order to run FLIMfit you will need the MATLAB Compiler Runtime (MCR) which enables the execution of compiled MATLAB applications or components on computers that do not have MATLAB installed. It can be downloaded from MATLAB Compiler Runtime (MCR). The Windows Installer will automatically install the MCR.
